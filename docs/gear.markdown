Specs:

- Processor: Intel Core i7-4770HQ @ 3.40GHz (4 Cores / 8 Threads)
- Motherboard: Apple Mac-06F11FD93F0323C5 (187.0.0.0.0 BIOS)

- Memory: 16384MB
- Disk: 251GB APPLE SSD SM0256
